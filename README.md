🏥 NFHS-5 Health Data Analysis and Geographic Visualization
📌 Overview
This project analyzes the National Family Health Survey (NFHS-5) state-level dataset to explore key public health indicators across India. Using Python, Pandas, Matplotlib, and Folium, the project performs data cleaning, exploratory data analysis (EDA), and interactive geographic visualization to compare health trends among different states.

The analysis focuses on indicators such as obesity, anaemia, high blood sugar, and hypertension, helping identify regional health disparities through statistical analysis and interactive maps.

✨ Features
📊 Data cleaning and preprocessing of the NFHS-5 dataset
📈 Correlation analysis between health indicators
📉 Grouped bar charts and scatter plots using Matplotlib
🗺️ Interactive India choropleth map using Folium
📍 State-wise health indicator visualization with interactive markers
💾 Exportable interactive HTML dashboard

# Mini-Project-Samsung-Innovation-Campus
Analyze NFHS-5 state-level health data using Python, Pandas, Matplotlib, and Folium. The project performs data cleaning, exploratory data analysis, and interactive geographic visualization of obesity, anaemia, high blood sugar, and other health indicators across Indian states.

This project analyzes the National Family Health Survey (NFHS-5) state-level dataset to study key public health indicators across India. Using Python, Pandas, Matplotlib, and Folium, the project cleans and processes health data, performs exploratory data analysis, and visualizes trends in obesity, anaemia, high blood sugar, and hypertension. It includes correlation analysis, grouped bar charts, scatter plots, and an interactive choropleth map that highlights regional health patterns across Indian states. The project demonstrates data preprocessing, statistical analysis, and geographic visualization techniques to support better understanding of health disparities and public health decision-making.


**Features**
Data cleaning and preprocessing using Pandas
Correlation analysis of health indicators
Grouped bar chart and scatter plot visualization
Interactive India choropleth map using Folium
State-wise health indicator comparison
Exportable interactive HTML dashboard

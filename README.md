# 🏥 NFHS-5 Health Data Analysis and Geographic Visualization

## 📌 Overview

This project analyzes the **National Family Health Survey (NFHS-5)** state-level dataset to explore key public health indicators across India. Using **Python**, **Pandas**, **Matplotlib**, and **Folium**, the project performs data cleaning, exploratory data analysis (EDA), and interactive geographic visualization to compare health trends across different states.

The analysis focuses on **obesity, anaemia, high blood sugar, and hypertension**, helping identify regional health disparities through statistical analysis and interactive maps.

---

## ✨ Features

- 📊 Data cleaning and preprocessing of the NFHS-5 dataset
- 📈 Correlation analysis between health indicators
- 📉 Grouped bar charts for state-wise comparison
- 🔍 Scatter plot visualization of health indicators
- 🗺️ Interactive India choropleth map using Folium
- 📍 Interactive state-wise markers with health summaries
- 💾 Exportable HTML dashboard

## 📂 Project Workflow

1. Load the NFHS-5 state-level dataset.
2. Clean and preprocess health indicator data.
3. Aggregate state-wise health statistics.
4. Perform exploratory data analysis (EDA).
5. Generate visualizations using Matplotlib.
6. Build an interactive geographic dashboard using Folium.
